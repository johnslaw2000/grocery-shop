# Grocery Shop

A small e-commerce web app for managing a grocery store inventory.
Users can sign in, browse products, add new items, edit prices, and remove products.

## Stack

- **Backend:** Python (Flask)
- **Database:** PostgreSQL
- **Containerization:** Docker + Docker Compose
- **Web server:** Nginx (reverse proxy)
- **CI/CD:** GitHub Actions
- **Hosting:** Ubuntu Server

## Status

Work in progress — DevOps learning project.
